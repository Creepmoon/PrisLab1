# Описание программы (ГОСТ-ориентированное)

## Назначение
EduSphere — образовательная клиент-серверная платформа с микросервисной архитектурой для хранения материалов, оценки успеваемости, генерации индивидуальных учебных планов и AR-сессий.

## Условия функционирования
- Серверная часть: Linux/Windows/macOS, Node.js 20+
- Клиентская часть: веб-интерфейс и мобильный AR-клиент (проектный контур)

## Логическая структура
- API Gateway
- Auth Service
- Content Service
- Gradebook Service
- Plan Builder Service
- Analytics Service
- AR Session Service

## Используемые алгоритмы
- HMAC-подпись токенов
- AES-256-GCM шифрование материалов
- Правил-ориентированная генерация рекомендаций и оценка точности
