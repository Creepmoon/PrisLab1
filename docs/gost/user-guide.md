# Руководство пользователя

## Вход и роли
- Ученик: просмотр оценок, планов и материалов
- Преподаватель: выставление оценок, создание материалов, правка планов
- Администратор: экспорт/удаление данных и управление доступом

## Базовые сценарии
1. Зарегистрироваться через `POST /auth/register`
2. Войти через `POST /auth/login` и использовать Bearer токен
3. Создать материал (`POST /materials`) от роли преподавателя
4. Выставить оценку (`POST /grades`)
5. Сгенерировать индивидуальный план (`POST /plans/generate`)
6. Проверить AR-совместимость (`POST /ar/check-device`)

## Формат справки
- Интерактивная справка: `apps/web/index.html`
- PDF-версия: формируется из этого документа в контуре поставки
